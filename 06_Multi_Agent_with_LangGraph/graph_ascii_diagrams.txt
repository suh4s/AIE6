===================================
ASCII Diagram: Verification Graph
===================================

+---------------------------------+
| ENTRY: supervisor               |
| (VerificationSupervisor Agent)  |
| - Receives draft context        |
| - Decides next step (FactCheck)|
+---------------------------------+
              |
              | 1. Route to FactChecker
              V
+---------------------------------+
| FactChecker Node                |
| - Reads summary from state      |
| - Reads 'x_draft.txt'           |
| - Compares facts                |
| - Returns AIMessage:            |
|   "ACCURATE" or "INACCURATE..." |
+---------------------------------+
              |
              | 2. Result returns to supervisor
              V
+---------------------------------+
| supervisor                      |
| - Examines FactChecker's output |
| - Decides next step             |
+---------------------------------+
         |        |
         |        +----[ Fact Check FAILED ("INACCURATE") ]-->+---------------------------------+
         |                                                    | Route to FINISH                 |
         |                                                    | (Triggers aggregation)          |
         |                                                    +---------------------------------+
         |                                                                  |
         |                                                                  V
         |        +----[ Fact Check PASSED ("ACCURATE") ]--->+---------------------------------+
         |        |                                         | Route to X_StyleChecker         |
         |        |                                         +---------------------------------+
         |        |                                                       |
         |        |                                                       V
         |        |                                         +---------------------------------+
         |        |                                         | X_StyleChecker Node             |
         |        |                                         | - Reads 'x_draft.txt'           |
         |        |                                         | - Checks length, tone, hashtags |
         |        |                                         | - Returns AIMessage:            |
         |        |                                         |   "STYLE OK" or "STYLE ISSUE..."|
         |        |                                         +---------------------------------+
         |        |                                                       |
         |        |<----------------[ Result returns to supervisor ]------+          
         |        |
         |        +----[ After Style Check ]----------------->+---------------------------------+
         |                                                    | Route to FINISH                 |
         |                                                    | (Triggers aggregation)          |
         |                                                    +---------------------------------+
         |                                                                  |
         V                                                                  V
+---------------------------------+                     +---------------------------------+
| aggregate_results Node          |-------------------->| END                             |
| (Aggregates checks, sets state) |                     +---------------------------------+
+---------------------------------+

ASCII Diagram: Final Graph (End-to-End Flow)
===================================

+---------------------------------+           +---------------------------------+
| ENTRY: Research Node            |---------->| Supervisor Node                 |
| (runs run_research)             |           | (Wrapper + LLM Agent)           |
| - Fetches Summary               |           | - Receives state from nodes     |
| - Initializes State             |           | - Decides next route based on   |
+---------------------------------+           |   prompt, state (verified,    |
              |                               |   rev_count, comments)          |
              |<------------------------------+---------------------------------+
              |                                             |
(Control returns to Supervisor)                             | Route based on
              |                                             | Supervisor's 'next' output
              V                                             V
+---------------------------------+           +---------------------------------+
| Authoring Node                  |<----------| Supervisor Node                 |
| (runs run_authoring)            | Routes to | (Output: next="Authoring")      |
| - Creates/Revises Draft         | Authoring |                                 |
+---------------------------------+           +---------------------------------+
              |
              |<------------------------------+
              |                               |
(Control returns to Supervisor)               |
              |                               |
              V                               V
+---------------------------------+           +---------------------------------+
| Verification Node               |<----------| Supervisor Node                 |
| (runs run_verification)         | Routes to | (Output: next="Verification")   |
| - Runs Fact & Style Checks      | Verify    |                                 |
+---------------------------------+           +---------------------------------+
              |
              |<------------------------------+
              |                               |
(Control returns to Supervisor)               |
              |                               |
              V                               V
+---------------------------------+           +---------------------------------+
| END                             |<----------| Supervisor Node                 |
| (Graph Execution Finishes)      | Routes to | (Output: next="FINISH")         |
|                                 | FINISH    |                                 |
